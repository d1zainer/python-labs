
## Блок 1. Синтаксис Python

**Синтаксис**

Синтаксис — это набор правил, по которым язык понимает наш код.
Если мы нарушаем правила синтаксиса, программа просто не запускается. Она не «догадывается», что мы имели в виду. Компьютер не интерпретирует намерения — он выполняет строго то, что написано.

Python создавался с идеей читаемости. Его философия в том, что код должен быть понятен человеку. Поэтому многие конструкции выглядят почти как обычный текст.

Главная особенность Python — это отступы.

В других языках блоки кода выделяются фигурными скобками. В Python блок кода определяется отступом. Отступ — это не оформление и не стиль. Это часть синтаксиса. Если отступ сделан неправильно, программа выдаёт ошибку.

Посмотрим пример.

```python
x = 5

if x > 3:
    print("Больше трёх")
```

Разберём, что здесь происходит.

Первая строка создаёт переменную `x` и присваивает ей значение 5.
Оператор `=` — это оператор присваивания. Мы не сравниваем, а записываем значение в переменную.

Вторая строка — условие. Конструкция `if` означает «если».
Если выражение `x > 3` истинно, то выполняется блок кода ниже.

Строка `print("Больше трёх")` имеет отступ. Это означает, что она относится к условию. Она выполнится только если условие истинно.

Если убрать отступ, Python выдаст ошибку, потому что нарушится структура блока.

Вывод простой: отступы в Python — это часть языка. Стандартный отступ — четыре пробела.

**Базовые операции**

```python
a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a / b)
print(a // b)
print(a % b)
```

Здесь показаны основные арифметические операции.

`+` — сложение 
`-` — вычитание
`*` — умножение
`/` — обычное деление, результатом будет число с плавающей точкой
`//` — целочисленное деление, дробная часть отбрасывается
`%` — остаток от деления

Оператор `%` часто используется для проверки чётности числа. Если число делится на 2 без остатка, значит оно чётное.

**Ввод данных.**

Программа может не только выполнять вычисления, но и получать данные от пользователя.

```python
name = input("Введите имя: ")
print("Привет,", name)
```

Функция `input()` останавливает выполнение программы и ждёт, пока пользователь введёт данные.
Важно помнить: `input()` всегда возвращает строку.

Если нам нужно получить число, строку нужно преобразовать:

```python
age = int(input("Введите возраст: "))
print(age + 5)
```

Функция `int()` преобразует строку в целое число.

Если вместо числа введут текст, программа завершится с ошибкой. Язык не угадывает, что мы хотели сделать.

**Булева логика**

В программировании всё сводится к двум значениям: `True` и `False`. Это логический тип данных.

Рассмотрим примеры сравнений:

```python
a = 5
b = 10

print(a > b)
print(a < b)
print(a == b)
print(a != b)
```

`>` — больше
`<` — меньше
`==` — равно
`!=` — не равно

Очень важно не путать `=` и `==`.
`=` — присваивание.
`==` — сравнение.

Теперь логические операторы.

```python
x = 7

if x > 5 and x < 10:
    print("Число в диапазоне")
```

`and` означает «и». Оба условия должны быть истинными.
`or` означает «или». Достаточно одного истинного условия.
`not` — отрицание.

Эти операторы позволяют строить более сложные логические проверки.

## Блок 2. Типы данных в Python

Python — язык с динамической типизацией. Это значит, что нам не нужно заранее объявлять тип переменной. Тип определяется автоматически по значению.


**Числовые типы**

В Python есть два основных числовых типа:

int — целые числа
float — числа с плавающей точкой

Пример:

```python
a = 10
b = 3.14

print(type(a))
print(type(b))
```

Функция type() показывает тип объекта.

int используется для целых чисел.
float используется для дробных чисел.

**Строки (str)**

Строка — это текст.

```python
name = "Иван"
message = 'Привет'

print(type(name))
print(len(name))
```

Строки можно создавать в двойных или одинарных кавычках.

Функция len() возвращает длину строки.

Строки можно объединять:

```python
first_name = "Иван"
last_name = "Иванов"

full_name = first_name + " " + last_name
print(full_name)
```

Важно понимать: строки нельзя складывать с числами без преобразования типов.

```python
age = 20
print("Возраст: " + str(age))
```

Здесь мы явно преобразуем число в строку с помощью str().

**Логический тип (bool)**

Логический тип имеет только два значения: True или False

```python
is_active = True
print(type(is_active))
```

Обычно bool получается в результате сравнений:

```python
print(5 > 3)
print(10 == 2)
```

Логические значения используются в условиях.

---

**Списки (list)**

Список — это упорядоченная коллекция элементов.

```python
numbers = [1, 2, 3, 4, 5]
print(type(numbers))
```

К элементам списка можно обращаться по индексу:

```python
print(numbers[0])
print(numbers[2])
```

Индексация начинается с нуля.

Можно добавлять элементы:

```python
numbers.append(6)
print(numbers)
```

Можно изменять элементы:

```python
numbers[0] = 100
print(numbers)
```

Списки изменяемые. Это важно.

**Кортежи (tuple)**

Кортеж похож на список, но он неизменяемый.

```python
coordinates = (10, 20)
print(type(coordinates))
```

Попытка изменить элемент вызовет ошибку:

```python
coordinates[0] = 50  # Ошибка
```

Кортежи используются, когда данные не должны изменяться.

**Словари (dict)**

Словарь хранит данные в формате ключ — значение.

```python
person = {
    "name": "Иван",
    "age": 20
}

print(type(person))
```

Доступ к значению происходит по ключу:

```python
print(person["name"])
```

Можно добавить новый элемент:

```python
person["city"] = "Москва"
print(person)
```

Словарь удобен для хранения структурированных данных.

**Множества (set)**

Множество — это коллекция уникальных элементов.

```python
numbers = {1, 2, 3, 3, 4}
print(numbers)
```

Повторяющиеся элементы автоматически удаляются.

Множества используются, когда важна уникальность.

**Преобразование типов**

Иногда нужно изменить тип данных.

```python
x = "10"
y = int(x)

print(type(y))
```

Основные функции преобразования:

int() — в целое число
float() — в дробное
str() — в строку
bool() — в логическое значение

Важно понимать: не каждое значение можно преобразовать.
Например, строку "abc" нельзя преобразовать в int.

**Динамическая типизация**

В Python переменная может менять тип:

```python
x = 5
print(type(x))

x = "теперь строка"
print(type(x))
```


## Блок 3. Условия и циклы

**Условия (if, elif, else)**

**Пояснение:** используем конструкции `if`, `elif` и `else`, чтобы проверять разные варианты.

```python
x = 10

if x > 0:
    print("Число положительное")
elif x < 0:
    print("Число отрицательное")
else:
    print("Число равно нулю")
```

* `if` — проверяет условие
* `elif` — проверяет другое условие, если `if` было ложным
* `else` — выполняется, если все предыдущие условия ложные»


**Логические операторы**

**Пояснение:** условия можно комбинировать с помощью `and`, `or`, `not`.

```python
x = 7

if x > 5 and x < 10:
    print("Число в диапазоне от 5 до 10")
```

**Цикл `while`**

**Пояснение:** цикл `while` выполняется, пока условие истинно.

```python
count = 0

while count < 3:
    print("Счётчик:", count)
    count += 1  # увеличиваем на 1
```

«Объяснение:

* Важно менять переменную внутри цикла, иначе он станет бесконечным»

**Цикл `for`**

**Пояснение:** цикл `for` удобен для перебора элементов последовательности.

```python
# перебираем числа от 0 до 4
for i in range(5):
    print("Итерация:", i)
```

* `range(5)` даёт последовательность 0,1,2,3,4
* Цикл выполняется по каждому элементу»

## Блок 4. Функции в Python

Функция — это кусок кода, который выполняет определённое действие и может возвращать результат.
Функции нужны для того, чтобы не писать один и тот же код несколько раз, а структурировать программу.

**Объявление функции**

Функция объявляется с помощью ключевого слова `def`.

Пример:

```python
def greet():
    print("Привет!")
```

Чтобы функция сработала, её нужно вызвать:

```python
greet()
```

Разбор:

* `def` — объявление функции
* `greet` — имя функции
* круглые скобки — место для аргументов
* двоеточие и отступ — тело функции
* print — действие функции

**Функции с аргументами**

Функция может принимать данные от пользователя:

```python
def greet(name):
    print("Привет,", name)
```

Вызов:

```python
greet("Иван")
greet("Мария")
```

Разбор:

* `name` — аргумент функции
* Каждый раз можно передавать разные значения
* Функция использует эти значения внутри тела

**Функции с возвратом значения**

Иногда функция должна что-то посчитать и вернуть результат.

```python
def add(a, b):
    return a + b
```

Вызов:

```python
result = add(5, 3)
print(result)
```

Разбор:

* `return` — возвращает результат из функции
* После return функция завершает выполнение
* Если return нет, функция возвращает None

**Значения по умолчанию**

Можно задавать значения аргументов по умолчанию:

```python
def greet(name="Гость"):
    print("Привет,", name)
```

Вызовы:

```python
greet("Иван")  # Привет, Иван
greet()        # Привет, Гость
```

Так функции становятся гибкими.


**Область видимости переменных**

В Python есть локальные и глобальные переменные.

```python
x = 10  # глобальная

def test():
    y = 5  # локальная
    print(y)

test()
print(x)
```

Разбор:

* Переменные, объявленные внутри функции — локальные, они живут только в функции
* Переменные вне функции — глобальные, их видят все функции (но менять глобальные переменные из функции лучше осторожно)

**Анонимные функции (lambda)**

Для маленьких функций можно использовать `lambda`:

```python
square = lambda x: x ** 2
print(square(5))
```

Разбор:

* `lambda x: x ** 2` — функция, которая возводит число в квадрат
* Удобно для передачи функции в другую функцию, например, `map` или `filter`


## Блок 5. ООП (Объектно-ориентированное программирование) — теория


Объектно-ориентированное программирование (ООП) — это подход к программированию, где всё в программе рассматривается как **объекты**.

Объект — это сущность, которая содержит **данные** (атрибуты) и **поведение** (методы).

Простейший пример: человек.

* Атрибуты: имя, возраст, рост
* Методы: идти, говорить, есть

ООП помогает моделировать реальный мир в коде. Вместо того чтобы писать сотни отдельных функций, мы создаём объекты, которые знают, что им нужно делать, и хранят свои данные внутри себя.

---

## Основные понятия ООП

1. **Класс**
   Класс — это шаблон, чертёж объекта. Он описывает, какие данные и методы будут у объекта.
   Пример: класс `Person` описывает человека, но сам по себе ещё не человек.

2. **Объект (экземпляр класса)**
   Объект — это конкретная реализация класса.
   Пример: `p = Person("Иван", 20)` — это объект класса `Person`.

3. **Атрибуты**
   Данные объекта.
   Пример: `p.name` или `p.age`

4. **Методы**
   Функции внутри класса, которые описывают поведение объекта.
   Пример: `p.greet()` — метод, который выводит приветствие.

5. **Инкапсуляция**
   Сокрытие внутренней реализации объекта.
   Другие объекты должны использовать только методы и публичные атрибуты, а не менять внутренние данные напрямую.
   В Python это реализуется соглашением через `_` (protected) или `__` (private).

6. **Наследование**
   Позволяет создавать новый класс на основе существующего.
   Новый класс получает все атрибуты и методы родителя, и может добавлять свои.

7. **Полиморфизм**
   Способность объектов разных классов иметь одинаковые методы с разной реализацией.
   Например, метод `draw()` может по-разному работать для объектов `Circle` и `Square`.

## Почему ООП важно

* Делает код структурированным и понятным
* Позволяет повторно использовать классы в разных проектах
* Облегчает поддержку больших программ
* Помогает моделировать сложные системы через объекты

## ООП в Python

**Класс и объект**

**Пояснение:** Класс — шаблон объекта. Объект хранит данные и умеет что-то делать.

```python
# создаём класс Person
class Person:
    def __init__(self, name, age):
        self.name = name  # атрибут имя
        self.age = age    # атрибут возраст

    def greet(self):
        print(f"Привет, меня зовут {self.name} и мне {self.age} лет")
```

```python
# создаём объект p1 и вызываем метод
p1 = Person("Иван", 20)
p1.greet()
```

**Простейшая инкапсуляция**

**Пояснение:** Приватный атрибут нельзя менять напрямую. Используем методы класса.

```python
class Person:
    def __init__(self, name, age):
        self.__name = name  # приватный атрибут
        self.age = age

    def greet(self):
        print(f"Привет, меня зовут {self.__name} и мне {self.age} лет")

    def set_name(self, new_name):
        self.__name = new_name
```

```python
# создаём объект и меняем имя через метод
p2 = Person("Сергей", 25)
p2.greet()
p2.set_name("Игорь")
p2.greet()
```
**Мини-класс Student**

**Пояснение:** Храним данные студента (имя, оценки) и считаем средний балл.

```python
class Student:
    def __init__(self, name):
        self.name = name
        self.grades = []

    def add_grade(self, grade):
        self.grades.append(grade)

    def average(self):
        if self.grades:
            return sum(self.grades) / len(self.grades)
        else:
            return 0
```

```python
# создаём студента, добавляем оценки и выводим средний балл
s1 = Student("Иван")
s1.add_grade(5)
s1.add_grade(4)
print(f"{s1.name} - средний балл: {s1.average()}")
```
**Lambda-функция (опционально)**

**Пояснение:** Короткая одноразовая функция.

```python
# функция для возведения числа в квадрат
```

## Блок 6. Подключение библиотек и модулей


**Пояснение:** стандартная библиотека уже идёт с Python. Например, модуль `math` для работы с числами.

```python
import math  # подключаем модуль math

# используем функцию для вычисления квадратного корня
print(math.sqrt(16))  # 4.0

# константа pi
print(math.pi)
```

**Пояснение:** можно импортировать только то, что нужно.

```python
from math import factorial

# вычисляем факториал числа
print(factorial(5))  # 120
```

**Пояснение:** можно давать псевдонимы библиотекам для удобства.

```python
import math as m

print(m.sqrt(25))  # 5.0
```

**Пояснение:** кроме стандартных, есть сторонние библиотеки, которые нужно устанавливать через `pip`.
Например, библиотека `random` для случайных чисел (стандартная), но принцип такой же: сторонние библиотеки устанавливаются отдельно.

```python
import random

# случайное число от 1 до 10
print(random.randint(1, 10))
```

* `import` подключает библиотеку целиком
* `from ... import ...` — подключает только нужные функции или классы
* Можно использовать псевдонимы (`as`)
* С помощью библиотек можно расширять функциональность Python без лишнего кода
